# shopping-mall-management-php
Responsive shopping mall management website built with PHP and Bootstrap, featuring product browsing, search, cart, order/payment flow, and admin management functions.
PHP와 Bootstrap을 활용해 제작한 반응형 쇼핑몰 판매관리 웹사이트입니다. 상품 목록 조회, 검색, 장바구니, 주문·결제, 관리자 기능까지 포함한 실습형 온라인 상점 프로젝트입니다.
